# FFLY Basic

一个基于 Go 语言开发的基础后台管理系统，提供用户管理、角色权限管理、API 日志等基础功能。

## 功能特性

- 用户管理
  - 用户注册/登录
  - 用户信息管理
  - 密码加密存储
  - 登录限流保护

- 角色权限管理
  - 基于 RBAC 的权限控制
  - 角色管理
  - 权限分配
  - 动态权限验证

- 系统功能
  - JWT 认证
  - API 访问日志
  - Redis 缓存支持
  - MySQL 数据存储

## 技术栈

- Go 1.20+
- MySQL 5.7+
- Redis 6.0+
- JWT
- Gin Framework
- GORM
